Code School

We run one day workshops in Meteor, Javascript and node. These days are for beginners and developers.

Our courses run on the weekends in the mornings. Each morning we will school you in the TABLEFLIP classroom, with talks, live coding demos, and amazing diagrams on the whiteboard. Then things get more relaxed and you can play with what you have learnt. You might have a project you are working on or you can chose from one of our pre-prepared exercises. We will come round and pair with you. We like to finish the weekend with a big phat show and tell, high fives and air punches. On both days we will provide a lunch for you. You will have the option to stay in the space and work on your project. Members of the TABLEFLIP team will be available during the afternoon.

Meteor

This course will have you building Meteor apps like a pro.

You will learn about the imperative programming style that Meteor uses, and all the different ways to harness the live reactive quality of Meteor. We will spend some time looking at flow-router and flow- layout, so you can confidently structure scalable apps. We will look at Meteor's data storage API. Finally we’ll finish the session by preparing your Meteor project to be downloaded as a native app on a smart phone

Love CSS

The course is all about loving css and laying out beautiful, responsive pages.

All the things we have learnt about making awesome websites that work for all the devices you can imagine are distilled into this course. We start with customising bootstrap and laying down some responsive ground rules. We’ll share our house style and philosophy on the build step, and show you how to use simple npm scripts to compile and optimise all your front-end assets so they snap down to the device like a ninja.

Node

This course will introduction you to node and some of the core programming patterns that we feel you cannot do without.

Javascript has matured into a full stack programming language. You can write javascript on the server and in the client’s browser. You will learn to write tests with tape and create modules for the server and client. You will be introduced to underscore and async, tools to help you build readable code that correctly controls the node process. We will then have you using your own modules to build from scratch a simple node server that runs a little website.

Arduino

Spend the day with us making actual things, actually flash and spin with javascript.

We build robots. We'll introduce you to the arduino prototyping board and the Johnny-five node library that you can use together to programme all the things. After a morning of instructions and experimenting with different physical components, you will spend the afternoon on assembling an awesome project of your choice. We’ll supply you with tools and materials to bring your ideas buzzing and flashing into life.
